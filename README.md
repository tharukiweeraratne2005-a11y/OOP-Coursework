# GearRentPro - Equipment Rental Management System

### Project Description
This is a professional photography equipment rental management system designed for a coursework submission. It handles branches, categories, equipment, and customer rentals.

### Default Login Credentials
- **Admin:** admin / 1234

### Database Configuration
1. The project includes a database backup file named `database.sql` (or inside the uploaded ZIP).
2. Import this `database.sql` into your MySQL server to create the tables and load the sample data.
3. Ensure the database connection settings in the source code match your local MySQL credentials.
